Hidden Code

It's a jQuery plugin.

Hidden Code allows you to activate an hidden code on your website (ex: the konami one)

Examples

You can see some examples on the examples folder.

Basic use

$(window).hiddenCode({
    callback: function() {
        alert('You found the hidden Code !');
    }
});
